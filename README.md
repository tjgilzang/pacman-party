# 정돈스돈스 팩맨

이 리포지토리는 간단한 HTML/JS 기반 **정돈스돈스 팩맨** 게임을 담고 있습니다. 플레이어는 화살표 키나 터치로 팩맨을 움직여 필드를 돌아다니며 점수를 쌓고, 유령과 충돌하면 목숨을 잃게 됩니다. 스페이스바를 눌러 언제든 다시 시작할 수 있어요.

## 특징
- 캔버스 기반 레트로 스타일 화면
- 벽과 펠릿 자동 생성
- 유령과 충돌 시 삶이 줄어드는 간단한 AI
- 점수, 목숨, 상태를 상단에 표시
- 오디오 피드백(펠릿 먹기/충돌) 포함
- 터치 스와이프 또는 키보드로 조작

## 실행
```bash
open pacman/index.html
```

또는 이 리포지토리를 GitHub Pages로 배포하여 링크로 공유할 수 있습니다.
